# project requirements
A data pipeline built with pandas and sql,
,using ETL to load a dataset, clean/transform it, and store output into a data warehouse.

## Software Requirements
- Python 3 +
- Pandas 
- SQL (SQLite/Postgres/Other)
- Data Warehouse
- Large dataset
- Logging (software/data lineage)
- Classes and modules
- Hosted on Github

## optional
- Numpy
- Matplotlib(or any other visualization)
- AWS (RDS or EC2)
- Advanced data formats(parquet)
- Daemon process (ideally with cron)
